HTW Saar Grade Tracker

A Python automation script that monitors the HTW Saar SIM portal and sends a Telegram notification whenever a new grade is published.

Description

The HTW Grade Tracker is a small automation tool that regularly checks the student portal of HTW Saar for newly published exam grades. The script logs into the SIM portal, downloads the transcript PDF and extracts the grade information automatically.

By comparing the extracted data with the previously saved state, the script detects newly added grades and immediately sends a Telegram notification to the user. This allows students to get notified instantly without manually checking the portal.

The tool is designed to run automatically on a schedule using a cron job and operates with a headless browser, making it suitable for deployment on servers or small automation environments.

Key Features

  • Automated Portal Login
    Logs into the HTW Saar SIM portal automatically using a headless browser session.
  • PDF Grade Extraction
    Downloads the transcript PDF and extracts grade information for automated processing.
  • Change Detection
    Compares the current grade state with previously stored results to detect newly published grades.
  • Telegram Notifications
    Sends instant notifications through Telegram whenever a new grade appears.
  • Cronjob Automation
    Designed to run automatically on a schedule using cron jobs.
