FIT Combiner

Java tool for merging multiple Garmin .fit activity files into a single combined workout file.

Description

FIT Combiner is a lightweight Java utility that merges multiple Garmin .fit activity files into a single continuous activity. It is especially useful when devices accidentally split recordings into several parts during long workouts.

The tool reads all records from the provided files, sorts them chronologically and writes a clean combined output file while preserving important activity metrics such as timestamps, heart rate, distance and speed.

The application uses the official Garmin FIT SDK and is packaged as a portable executable JAR file, allowing it to run on any system with a Java runtime.

Key Features

  • FIT File Merging
    Combines multiple Garmin .fit files into a single continuous activity recording.
  • Chronological Sorting
    Automatically orders all activity records by timestamp to ensure a consistent output file.
  • Metadata Preservation
    Keeps important data such as heart rate, speed, distance and timestamps intact during the merge.
  • Garmin FIT SDK Integration
    Uses the official Garmin FIT SDK for reliable decoding and encoding of activity data.
  • Portable CLI Tool
    Distributed as a standalone .jar file that can run on any system with a Java runtime.
