F1 Strategy Tool

A Formula 1 race strategy simulator to analyze tire strategies, pit stop windows and race pace.

Description

The F1 Insight Tool allows users to analyze historical Formula 1 data such as race results and session information. Based on this data, the application can generate visualizations and graphs to explore performance trends and race statistics.

The project originally started as a university semester project where the goal was to develop a command line tool in Java using Maven as the build system. After completing the CLI version, I extended the project by creating a simple web interface using TypeScript, HTML and CSS, and deployed the application with Docker on Render.

Key Features
